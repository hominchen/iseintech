關於我們

祁祁知科技成立於2015年6月，祁知科技初始成員來自工研院、資策會、研華科技等，並以「新產品開發」暨「影像自動化設備開發」為主要業務。針對日常工作與生活中無價值的重複性動作，提供自動化與數位化解決方案。

我們定位為專業的軟體開發公司，致力於為中小型製造業提供創新的企業資訊系統。我們的產品包括(1)自動光學缺陷檢測系統、(2)影像數量盤點系統、(3)影像安防系統、(4)AI趨勢預測系統等，能夠幫助客戶提高生產效率、降低成本、增加競爭力。

我們的產品不需要高額的預算導入，也不需要客戶有強大的資訊數位化能力，只要您有少量多樣的需求，以及願意嘗試新技術的心態，我們就能為您量身打造最適合您的解決方案。我們是您數位轉型的最佳夥伴！
