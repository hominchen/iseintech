01. 網頁設計常見問題

租用網站的優缺點
優點
設計費用較低 租用型的網站一般以"年"為單位繳費，費用可以從3,000~30,000不等。由於網站的服務商已經開發好模組，因此在費用上較為低。
缺點
沒有網站所有權 租用型網站的費用較低，但客戶並沒有網站的所有權，在空間的大小上也有限制；如果業者後來無法繼續提供租用的服務，客戶就會受到損失。租用型網站採用每年繳費的方式，對於想要長期經營網站的客戶來說，在費用支出上並不會比購買的網站要來的划算。
版面較制式 租用型網站是業者預先開發好的模組，所以在版面設計上無法根據客戶的需求客製化，除非業者有提供客製化美編設計的租用型方案，否則客戶只能依業者所提供的版型選擇使用，因此在整體的設計風格上較固定無特色。
功能受限制 租用型網站的使用功能是業者預設好的，客戶只能按照業者所提供的功能作使用，如果要其他常見的附加功能，就需要另外付費購買；但是如果客戶要的功能是較特別的，租用型的網站可能就無法符合。
不提供修改 由於租用型的網站是共用的租用平台，所以無法根據每個客戶的需求做出調整，這常常導致客戶使用上的不方便。
套版網站的優缺點
優點
設計費用較低 套版網站是業者事先設計好的版型，所以比客制化網站便宜，但也比租用型網站貴一些。
一次性繳清 套版網站只要付一次費用，就能擁有網站的所有權。但有些套版網站的業者會規定客戶必須使用他們的空間，並每年收取空間費用，以降低製作的成本。
缺點
版面設計制式 套版網站和租用型網站一樣，都是業者事先設計好的版型，所以無法客製化版面設計，除非業者提供客製化美編設計的套版型方案，不然客戶只能選擇業者提供的套版樣式，因此在設計風格上較缺乏特色。
功能受限制 套版網站是業者預先開發好的版型，所以客戶只能使用業者提供的功能。而且有些業者開發的模組功能太複雜，讓客戶不知道怎麼操作，反而增加了客戶使用上的困擾。
無法任意修改 套版網站只能做些微的修改或者不能修改，客戶修改時還要付額外的費用，如果修改的範圍大，費用可能跟客制化網站一樣高。
客製化網站優缺點
優點
取得網站所有權 只要付一次費用，就能擁有網站的所有權。有些專業的網頁設計還會提供可修改的原始檔，讓客戶可以備份或修改網站。
依需求設計網站風格 客戶可以根據自己的需求設計網頁的風格，不但能滿足客戶的期望，也能展現網頁的特色。
功能不受限制 可以根據自己的需求設計後端的管理系統功能，讓管理網頁更方便和高效。
可任意修改 客制化網站的修改很靈活，可以隨時根據客戶的要求做調整，不受既有的功能模組的限制。
缺點
設計費用較高 客制化網站是根據客戶的需求訂做的，所以建置費用通常比租用型和套版型的網站高一些。
設計的時間較長 客制化網站是按照客戶的需求訂做的，所以在建置過程中需要花更多的精力和時間，因此建置的時程會比其他兩種類型的網站長一些。

  1. 網頁設計： 如果您想要設計一個符合企業形象和功能的網站，您可能需要委託專業的網頁設計公司或網站開發團隊來幫您。這些服務的費用會依據您的需求和工作量而有所差異，一般來說，網頁設計的費用會根據「網站架構」和「網站功能」來報價，費用通常從 1~2 萬元到數十萬元不等。
  2. 網站空間： 當您的網頁設計完成後，您需要有一個網站空間來存放您的網頁。您可以選擇自行架設伺服器，也就是自己買一台電腦當伺服器，並全年開機運作。這種方式的費用較高，約在 3 萬元到數十萬元之間。或者，您也可以選擇使用共享主機、虛擬私有伺服器（VPS）或專用伺服器等不同的託管方案，也就是租用「虛擬主機」來存放您的網頁。這些方案的費用會因為提供商和服務方案而有所不同，取決於伺服器性能和資源的需求。一般來說，虛擬主機租用費用約在每年 NT$6,000 至 NT$12,000 之間。需要注意的是，虛擬主機的連線品質和軟硬體效能會直接影響到網站的執行速度。
  3. 網域名稱： 當您有了網頁和網站空間後，最後就是要有一個屬於自己網站的「網址」。您必須去申請網域名稱，才能有一個固定的網址。目前網域名稱申請費用約為每年 NT$800 左右。
  4. 網站維護和更新： 網站不是一次性的產品，而是需要持續地維護、更新和保護。您可能需要考慮花費資源來定期更新內容、修復漏洞、備份數據等。這些費用可能包括自己的時間和/或外包給維護服務提供商的費用。
  5. 金物流或第三方串接： 除了以上基本的網站架設費用外，還有一些其他可能產生費用的項目，如 SSL、金流串接、物流串接、第三方插件或工具、網站分析等。

自己架設網站的費用會根據需求的不同而有所變化。有些開源的免費軟體和服務可以幫助降低成本，但可能需要更多的技術知識和自行管理成本。建議在開始建立網站之前，評估您的需求、預算和技術能力，並考慮諮詢網頁設計專業人士以獲得更具體的網站架設費用評估。

如果您不想自己開發設計網站，您可以選擇一些「免費網站」的平台，讓您可以快速建立一個簡單的網站。但是這些免費網站平台通常有一些限制或缺點，例如網站功能或流量有限制、強制觀看廣告、資安有問題等。而且使用這些免費網站平台還是需要有一些網頁設計基礎才能上手。


網站改版的定義和費用因情況而異，以下是幾種常見的改版類型和相關的費用考量：

  1. 網站只需要局部調整內容或功能 這種情況不算是真正的改版，而是網站的「修改或更新」。費用方面，只會根據修改或更新的內容來收取。但是，如果您找的不是原來製作網站的設計公司，那麼費用可能會比較高，因為接手的設計公司或程式設計師可能需要花更多時間來了解和修改別人做的網頁或程式。
  2. 網站整體架構和功能不變，只要調整視覺風格 這種情況不需要重做網站，只要將新設計的前台版面套用到現有的網站前台系統上即可。費用方面，只會根據新設計的前台版面來收取。但是，如果您找的不是原來製作網站的設計公司，那麼費用可能也會比較高，因為套用新版面可能需要調整現有的系統程式。
  3. 網站整體視覺風格不變，但系統功能和內頁需要重做 這種情況幾乎等於重新製作網站，因為網站系統的開發時間和成本通常遠高於網頁的美工設計。費用方面，只能省下原來網站視覺設計的部分。

如果您想要節省網站改版的費用，以下是一些建議：

  1. 精確地分析網站需求： 在開始改版之前，仔細地分析網站需要改善和修改的地方，確定改版的目標和範圍。這樣可以避免浪費時間和資源在不必要或無效的修改和開發上。
  2. 利用現有資源和工具： 盡量使用現有的網站結構和資源，減少重新製作的工作量。同時，使用現成的網站模板和開源工具，可以降低設計和開發的難度和成本。
  3. 專注於關鍵功能和優先事項： 將注意力集中在網站最重要和最急迫的功能和事項上，以達成核心目標。這樣可以避免過度設計和過度複雜化網頁和功能，同時節省開發時間和成本。
  4. 測試和驗收： 在改版完成之前，進行充分的測試和驗收，以確保網站運作正常並符合預期的需求。這可以減少後期修正和調整所需的時間和成本。

如果您的預算有限，您可能會考慮選用套版型或租用型的網站方案。這些方案通常比較便宜，而且可以讓您快速建立您的網站。 但是，在選擇套版型或租用型的網站方案之前，您需要注意以下幾點：

  1. 功能需求： 選擇一個符合您的網站功能需求的方案。檢查模版或租用方案提供的功能列表，並確認它們是否包含您需要的重要功能，例如在線購物、聯繫表單、部落格等。
  2. 客製選項： 了解模版或租用方案是否可以讓您自行調整一些設定。您可能想要修改一些顏色、字體、佈局等，以讓您的網站更符合您的品牌形象。
  3. 技術支持： 確認模版或租用方案是否有提供技術支持。如果您在使用網站時遇到任何問題或困難，您可以向他們尋求協助。 另一方面，客製型的網站方案並不一定比套版型或租用型的網站方案貴很多。客製型的網站方案通常會比較貴，是因為客製型的系統功能通常比較多或比較複雜，如果您需要很多系統功能，例如購物型的網站（需要商品管理、會員管理、訂單管理、報表管理等），而套版型或租用型的網站方案剛好能滿足您的需求，那麼選用套版型或租用型的網站方案肯定是比較划算的。

但是，如果您要建立的只是一個簡單的形象型網站，主要是展示公司介紹、最新訊息、商品介紹等，那麼您也可以考慮客製型的網站方案。如果客製型和套版型或租用型的網站方案價格差不多，或者在您可以接受的範圍內，那麼選用客製型的網站方案會有更多的優勢。客製型的網站方案可以讓您擁有更高品質、更有特色、更符合需求、更具彈性和擴充性的網站。

04. 參考資源

現在，架設網站並不困難，您可以利用許多現成的資源和工具來快速建立您的網站。iSeinTech就是一個協助您開始電子商務的夥伴，讓您可以輕鬆開始您的生意。

WordPress是一個開源的內容管理系統（CMS），它可以讓您在不需要寫任何程式碼的情況下，輕鬆地建立和管理您的網站。您可以使用WordPress來建立各種類型的網站，例如部落格、商店、社群、教育、媒體等等。

要開始使用WordPress，您需要以下幾個步驟：

  1. 註冊域名： 域名是您的網站的地址，例如weblai.co或wordpress.com。您可以在各種域名註冊商那裡購買您喜歡的域名，或者使用WordPress.com提供的免費域名。
  2. 選擇主機： 主機是您的網站的存放空間，它會影響您的網站的速度和安全性。您可以選擇自己租用主機，或者使用WordPress.com提供的免費或付費的主機服務。
  3. 安裝WordPress： 如果您選擇自己租用主機，您需要自己下載並安裝WordPress軟體到您的主機上。如果您選擇使用WordPress.com，則不需要這個步驟，因為WordPress已經預先安裝好了。
  4. 設定WordPress： 安裝好WordPress後，您需要設定一些基本的資訊，例如您的網站名稱、描述、語言、時區等等。您也可以選擇一個主題來決定您的網站外觀，以及一些插件來增加您的網站功能。
  5. 建立內容： 最後，您就可以開始建立和發佈您的網站內容了。您可以使用WordPress提供的方便的編輯器來創造文章、頁面、圖片、影片等等。

這就是WordPress的初學者教學文章，希望對您有幫助。

如果您想要了解更多詳細的教學，您可以參考官方網站 WordPress。


Bootstrap是一個開源的框架，它可以讓您使用HTML、CSS和JavaScript來建立響應式、行動優先的網站。Bootstrap提供了許多預先設計的元件、佈局和插件，讓您可以快速和容易地創造出美觀和功能豐富的網站。

要開始使用Bootstrap，您需要以下幾個步驟：

  1. 下載Bootstrap： 您可以從官方網站下載Bootstrap的最新版本，或者使用CDN來引用Bootstrap的檔案。
  2. 建立HTML檔案： 您需要建立一個HTML檔案，並在 head 區塊中加入Bootstrap的CSS檔案，以及在 body 區塊中加入Bootstrap的JavaScript檔案。您也可以選擇使用模板來快速建立基本的網頁結構。
  3. 使用Bootstrap元件： 您可以在HTML檔案中使用Bootstrap提供的各種元件，例如按鈕、導覽列、卡片等等。您只需要加入對應的class名稱，就可以套用Bootstrap的預設樣式和功能。
  4. 使用Bootstrap佈局： 您可以使用Bootstrap提供的格線系統和斷點來控制您的網頁在不同裝置上的排版和響應式。您也可以使用容器、排列和偏移來調整您的網頁間距和對齊。
  5. 使用Bootstrap插件： 您可以使用Bootstrap提供的JavaScript插件來增加您的網頁互動性和動畫效果。例如，您可以使用模態視窗、輪播圖、提示工具等等。您只需要加入一些data屬性或JavaScript程式碼，就可以啟用這些插件。

如果您想要了解更多詳細的教學，您可以參考官方網站 Bootstrap
